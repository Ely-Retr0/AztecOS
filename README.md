# AztecOS 
**The perfect balance between nature, development, and cybersecurity.**

AztecOS is a Debian-based Linux distribution, designed from the ground up to offer an efficient, highly customizable, and combat-ready working environment. Whether you are writing code or auditing networks, AztecOS gives you the tools you need without sacrificing performance or aesthetics.

## About the Project
The goal of AztecOS is to democratize access to a professional hacking and development environment. Many cybersecurity distributions are overloaded or difficult to use as a daily driver operating system. AztecOS solves this by offering a solid Debian base, guaranteeing stability, but with a minimalist approach that allows you to scale the system to your needs.

### Similarities with other distros:
* **Debian Base:** Shares the same stability, package management (`apt`), and security as distros like Kali Linux or Parrot OS.
* **Pentesting Orientation:** Comes preconfigured with industry-standard tools.

### Key differences:
* **Lightweight and Efficient:** We don't install bloatware. Only what is necessary for the system to "fly", ideal for compiling code or running brute-force scripts without bottlenecks.
* **"Nature & Tech" Aesthetics:** A unique visual design inspired by the jungle and Aztec iconography (earthy colors, jade green), breaking the mold of the classic "neon green on black".
* **Total Freedom:** Although it has a strong visual identity, the architecture allows the user to dismantle and customize every visual and functional corner of the distro.

## 💻 Installation Guide
*(Note: AztecOS is in active development. The download links for the .ISO image will be available soon).*

1. **Download the ISO:** Get the latest version of AztecOS from the *Releases* section in this repository.
2. **Create your Bootable USB:** Use tools like [BalenaEtcher](https://balena.io/etcher) or `dd` in Linux to flash the ISO onto a USB drive of at least 8GB.
3. **Boot from the USB:** Restart your computer, enter the BIOS/UEFI, and select the USB as the first boot device.
4. **Installation:** 
   * Select "Install AztecOS" from the boot menu.
   * Follow the graphical wizard to configure your language, time zone, and disk partitions.
   * Create your administrator user and wait for the installation to finish.
5. **Done!** Remove the USB and restart to enter your new environment.
